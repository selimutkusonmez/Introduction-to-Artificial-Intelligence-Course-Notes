Supervised Learning

It's mostly done with "labeled data" which means there is a output for every input. Used to solve regression and classification problems

Supervised Learning Steps

  • Data Collecting and Preparing
  • Model Choosing and Training
  • Model Evaluation
  • Model Configuration and Hyperparameter Optimization

Supervised Learning Examples

  • Market Segmentation
  • Stock price prediction
  • Disease diagnosis


Linear Regression

Linear Regression is a Supervised Learning method to predict constant output values

It's basicly used to model linear regression between input features and label

Linear Regression Examples

  • Rent Price Prediction
  • Stock price prediction


Logistic Regression

Logistic Regression is used in Machine Learning for Classification problems

It tries to predict a category of an input with features

Logistic Regression Examples

  • Spam Mail Classification


Decision Tree

Decision Tree is a Machine Learning algorithm to solve Classification and Regression problems

It's basicly creates a bunch of decision node from input data features and classifies it or does regression

Decision Tree Types

  • Random Forest
  • Gradient Boosting



Unsupervised Learning

It's mostly used in Data Exploring and Data Preprocessing steps

Reinforcement Learning

Reinforcement Learning is having an Artificial Intelligence model interact with an enviroment to learn how to accomplish a task

Reinforcement Learning Types

  • Non-Model Based Reinforcement Learning
  • Value Based Reinforcement Learning
  • Policy Based Reinforcement Learning
