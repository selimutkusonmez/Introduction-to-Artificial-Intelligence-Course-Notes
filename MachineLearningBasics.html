Supervised Learning

It's mostly done with "labeled data," which means there is an output for every input. Used to solve regression and classification problems.

Supervised Learning Steps

  • Data Collection and Preparation
  • Model Selection and Training
  • Model Evaluation
  • Model Configuration and Hyperparameter Optimization

Supervised Learning Examples

  • Market Segmentation
  • Stock Price Prediction
  • Disease Diagnosis


Linear Regression

Linear Regression is a Supervised Learning method used to predict constant output values.

It's basically used to model the linear relationship between input features and labels.

Linear Regression Examples

  • Rent Price Prediction
  • Stock Price Prediction


Logistic Regression

Logistic Regression is used in Machine Learning for Classification problems.

It tries to predict the category of an input based on features.

Logistic Regression Examples

  • Spam Mail Classification


Decision Tree

Decision Tree is a Machine Learning algorithm used to solve Classification and Regression problems.

It basically creates a bunch of decision nodes from input data features and classifies them or performs regression.

Decision Tree Types

  • Random Forest
  • Gradient Boosting



Unsupervised Learning

It's mostly done with "unlabeled data," which means there is no output for every input.

It's mostly used in Data Exploration and Data Preprocessing steps.

Clustering



Reinforcement Learning

Reinforcement Learning involves an Artificial Intelligence model interacting with an environment to learn how to accomplish a task.

Reinforcement Learning Types

  • Model-Based Reinforcement Learning
  • Model-Free Reinforcement Learning
  • Value-Based Reinforcement Learning
  • Policy-Based Reinforcement Learning
